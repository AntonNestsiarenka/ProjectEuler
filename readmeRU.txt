В модуле ProjectEuler представлены способы решения конкретных задач из сборника: Проект "Эйлер".
Все используемые функции реализованы в отдельном модуле euler_project. К каждой функции написана документация.

На данный момент в модуле ProjectEuler представлены решения 16 задач. К каждой задаче написано описание.
Для вывода результата в консоль определенной задачи требуется вызвать ее соответствующую функцию.
Функции для задач написаны в одном стиле "taskX", где X - это номер задачи. Для просмотра времени выполнения
кода задачи требуется поместить функцию вызова taskX() между строкой start_time = datetime.now() и print('{} {}'.format('Lead time:', datetime.now() - start_time)).
По возможности, решения новых задач будут добавляться в данный модуль.
